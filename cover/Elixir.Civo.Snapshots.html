2 @moduledoc """
3 Civo provides a backup service for Instances called snapshots. This takes
4 an exact copy of the instance's virtual hard drive. At any point an
5 instance can be restored to the state it was in when the snapshot was made.
6 These snapshots can also be used to build a new instance to scale
7 identically configured infrastructure.
8
9 As snapshot storage is chargeable, at any time these can be deleted. They
10 can also be scheduled rather than immediately created, and if desired
11 repeated at the same schedule each week (although the repeated snapshot
12 will overwrite itself each week not keep multiple weekly snapshots).
13 """
